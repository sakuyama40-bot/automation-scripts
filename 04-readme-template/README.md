# 04-readme-template — README 生成ツール

プロジェクト情報を対話式で入力すると、タイプに合った実用的な `README.md` を自動生成。

## 使い方

```bash
node readme-gen.js
```

## 対応タイプ（5種類）

| # | タイプ | 生成されるセクション |
|---|---|---|
| 1 | Node.js CLIツール | 概要・インストール・使い方・オプション一覧・ディレクトリ構成 |
| 2 | Express APIサーバー | 概要・セットアップ・起動・環境変数・エンドポイント一覧・ディレクトリ構成 |
| 3 | 汎用自動化スクリプト | 概要・セットアップ・実行・設定・処理の流れ・注意事項 |
| 4 | ライブラリ / パッケージ | 概要・インストール・使い方（コード例）・APIリファレンス |
| 5 | フルプロジェクト | 全セクション |

## 対話式入力の流れ

```
プロジェクト名 → 概要 → タイプ → Node.jsバージョン → ライセンス
→ タイプ別の追加質問（コマンド例・ポート・エンドポイント等）
→ 出力先ディレクトリ
```

## 既存ファイルの扱い

`README.md` が既にある場合は `README.md.bak` にバックアップしてから上書き。

## 依存パッケージ

なし（Node.js標準モジュールのみ）
